# Project-Day
# Python Learning Project

Учебный репозиторий с практикой по Python, разбитой по дням. Проект используется как часть менторского трека по переходу в backend и AI backend development.

## Что внутри

В репозитории собраны небольшие упражнения на базовые темы Python:

- работа со строками
- списки и числа
- функции
- словари и списки словарей
- простая обработка пользовательских данных

Также в проекте есть файл [PLAN.md](PLAN.md) с общим учебным планом на несколько месяцев.

## Структура проекта

```text
day1/  - базовые задачи: ввод данных, функции, подсчет слов
day2/  - работа с текстом и списками
day3/  - работа со словарями пользователей
day4/  - числа, фильтрация, среднее значение
day5/  - нормализация текста и простые утилиты
PLAN.md
```

## Как запускать

Нужен Python 3.10+.

Примеры запуска:

```bash
python day1/day1.py
python day2/day2.py
python day3/day3.py
python day4/day4.py
python day5/day5.py
```

Каждый файл запускается отдельно и выводит результат работы функций в консоль.

## Цель репозитория

Этот проект нужен для:

- закрепления основ Python на практике
- выработки привычки хранить учебный код в Git
- подготовки к следующим этапам: FastAPI, базы данных, Docker и AI backend

## Что можно улучшить дальше

- добавить единый стиль именования переменных
- вынести тестовые данные из исполняемых файлов
- добавить `requirements.txt` или описание окружения
- покрыть функции простыми тестами
- оформить каждый день как отдельную мини-задачу с описанием
