# CLI user_manager

## What does program?
This is program can add info about user, save her, find user by email or city. Info saved in JSON file and before reboot program, info will remain into file.

## How to launch program?
You need change directory on "day21" and launch python file this command: 'python3 day21.py'

## Which commands in menu program
1. add user
2. show user
3. find user by city
4. find user by email
5. save users
6. delete user by email
0. exit from program
